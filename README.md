# BTTrack

BTTrack — система для учёта строительных инструментов на строительных объектах. Приложение позволяет вести полный контроль за инструментами, работниками и объектами строительства, сокращая время на поиск и управление ресурсами, минимизируя потери и упрощая процессы передачи и списания оборудования.

## Основные возможности

* Учёт всех сотрудников с ролями, фотографиями и данными о трудоустройстве.
* Привязка сотрудников к строительным объектам.
* Ведение информации о бригадирах и их управлении объектами.
* Учёт инструментов с инвентарными номерами, QR-кодами, фотографиями и статусами.
* Проведение инвентаризаций с фиксацией наличия инструментов.
* Учёт передачи инструментов между объектами.
* Отслеживание списанных и повреждённых инструментов.

## Стек технологий

* **Frontend:** React
* **Backend:** Django + Django REST Framework
* **База данных:** PostgreSQL (или любая поддерживаемая Django)
* **Хранение медиа:** локальное или облачное хранилище

## Структура проекта

Проект разделён на несколько приложений для упрощения поддержки и расширения:

* `users` — управление пользователями, ролями, объектами и бригадирами.
* `tools` — учёт инструментов, статусов, фотографий, QR-кодов.
* `inventory` — проведение и хранение инвентаризаций, связь инструментов с проверками.
* `transfers` — учёт передачи инструментов между объектами.

## Установка

1. Клонируйте репозиторий:

```bash
git clone <URL_репозитория>
cd BTTrack
```

2. Создайте виртуальное окружение и установите зависимости:

```bash
python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate для Windows
pip install -r requirements.txt
```

3. Настройте `.env` файл с параметрами базы данных и медиа-хранилища.

4. Выполните миграции:

```bash
python manage.py migrate
```

5. Создайте суперпользователя:

```bash
python manage.py createsuperuser
```

6. Запустите сервер разработки:

```bash
python manage.py runserver
```

## Использование

* Администратор может управлять всеми объектами, пользователями и инструментами через Django Admin.
* Рабочие и бригадиры получают доступ к функционалу учёта инструментов и инвентаризаций через веб-интерфейс.
